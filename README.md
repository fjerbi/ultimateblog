# The Ultimate Blog

The Ultimate Blog is a package that allows you to integrate a whole blog system that contains:
* CRUD on posts
* Likes/Dislikes
* Commenting system
* Search posts
* Post user control system
* Notification and alerts system
* Author profile
* Responsive design
* Tags System
* Views count
* Subscribe to blog

## Future Features:
* Live chatroom
* Follow/Unfollow Author
* Admin panel
