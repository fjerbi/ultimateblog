# The Ultimate Blog

The Ultimate Blog is a package that allows you to integrate a whole blog system that contains:
-CRUD on posts
-Likes/Dislikes
-Commenting system
-Notification and alerts system
-Author profile
-Responsive design
-Tags System
-Views count

## Future Features:
-Live chatroom
-Follow/Unfollow Author
-Admin panel
